## 버전 관리 정책

### 하위 호환성
- 새 버전은 이전 버전 파일을 읽을 수 있어야 함
- 알 수 없는 레코드는 무시하고 처리 계속

### 상위 호환성
- 이전 버전은 새 버전의 기본 내용을 읽을 수 있어야 함
- 새로운 기능은 별도 레코드로 추가

### 버전 확인
- FileHeader의 version 필드로 파일 버전 확인
- 레코드 크기로 레코드 버전 구분
