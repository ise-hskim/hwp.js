## 개요

한글 문서 파일은 Microsoft의 복합 파일(Compound File) 구조를 기반으로 합니다. 이는 하나의 파일 안에 여러 개의 스트림(Stream)을 포함할 수 있는 파일 시스템과 유사한 구조입니다.

## 기본 구조

```
HWP 파일
├── FileHeader (파일 헤더)
├── DocInfo (문서 정보)
├── BodyText (본문)
│   ├── Section0
│   ├── Section1
│   └── ...
├── BinData (바이너리 데이터)
│   ├── BIN0001.jpg
│   ├── BIN0002.png
│   └── ...
├── PrvText (미리보기 텍스트)
├── PrvImage (미리보기 이미지)
├── DocOptions (문서 옵션)
├── Scripts (스크립트)
└── XMLTemplate (XML 템플릿)
```

## 주요 특징

1. **계층적 구조**: 스토리지와 스트림으로 구성된 트리 구조
2. **압축 지원**: 개별 스트림별로 압축 가능
3. **확장성**: 새로운 스트림 추가 가능
4. **호환성**: OLE 복합 파일 표준 준수
